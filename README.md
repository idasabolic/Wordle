Features

Fully playable in the terminal/console.
Gives feedback on guessed letters:
  Correct letter in correct position ✅
  Correct letter in wrong position ⚠️
  Incorrect letter ❌
Randomly selects a word from the official Wordle answer list.
Only allows official Wordle approved words as valid guesses.
Tracks the number of attempts and ends the game when the word is guessed or attempts run out.
